## 分类

有四种作用范围由小到大: (`private` < `空着不写` < `protected` < `public`)

| 修饰符 | 同一个类中 | 同一个包中其他类 | 不同包下的子类 | 不同包下的无关类 |
| ------------- | :-----------: | :----: | :----: | :----: |
| `private`      | √ |  |  |  |
| `default`      | √ | √ |  |  |
| `protected`      | √ | √ | √ |  |
| `public`      | √ | √ | √ | √|

---

## 使用规则

在实际开发中, 一般只是用`public`和`private`

- 成员变量私有
- 方法公开

> 特例: 如果方法中的功能类似于一种规则, 不希望别人使用, 那么这个方法就可以私有化
